
# SimpleOS


## Overview
This program aims to simulate the instruction cycle by generating a sequence of fibonacci numbers indefinitely until the user wishes it to stop 

## Setup

Compile SimpleOS

    $ javac simpleos.sys.OS.java 


Run SimpleOS

    $ java simpleos.sys.OS
